# طلب تصميم ثيم جديد — ابعت الملف ده لـ Claude Design

> انسخ المحتوى ده وابعته لـ Claude Design (أو أي أداة تصميم).
> لما يرجعلك الديزين، ابعتهولي وأنا أحوّله لثيم شغّال في الموقع.

---

## 🎯 البرومبت الأساسي (انسخه زي ما هو)

```
اعملي تصميم ثيم مميز ومختلف لمنصة بورتفوليو احترافية (زي Behance / Webflow).

المنتج: موقع بورتفوليو لمصممين ومصورين، يدعم عربي وإنجليزي (RTL/LTR).
الأقسام: Hero, About, Services/Expertise, Projects Gallery,
Experience, Testimonials, Clients, Contact.

المطلوب: ثيم له شخصية بصرية واضحة ومختلفة — مش مجرد تغيير ألوان.
اختار اتجاه واحد قوي ونفّذه بجرأة كاملة (مش محافظ).

أمثلة اتجاهات ممكن تختار منها:
- Brutalist (حواف حادة، تباين عالي، جريء)
- Luxury Editorial (مجلة فخمة، مساحات واسعة، تايبوجرافي درامي)
- Neo-Bold / Neo-Brutalism (ألوان صارخة، ظلال قوية، ترند)
- Swiss Minimal (شبكة صارمة، أبيض وأسود، دقة)
- Dark Cinematic (سينمائي، دراما ضوئية)

سلّمني بالتحديد:
1. لوحة ألوان بقيم HEX: accent, background, surface/card, text, muted/subtext
2. الخطوط: اسم خط العناوين + اسم خط النصوص (من Google Fonts تحديداً)
3. فلسفة الحواف (border-radius): حاد (0) / دائري خفيف / كبسولة
4. شكل الكروت: مصمت / زجاجي / إطار / بظل قوي — مع وصف
5. شكل الشريط العلوي (navbar): مصمت / شفاف / ضبابي
6. تصميم الـ Hero: وصف التخطيط (نص ضخم؟ صورة مقسومة؟ خلفية كاملة؟ بسيط؟)
7. شكل الأزرار: دائري / حاد / كبسولة
8. الإيقاع البصري: المسافات كبيرة (فخم) ولا مضغوطة (كثيف)؟
9. أي عناصر مميزة: خطوط فاصلة، ظلال، حركات، أرقام، badges، إلخ
10. Mockup / صورة للصفحة الرئيسية عشان أشوف الشكل

خليه Responsive ويشتغل على الموبايل.
لو تقدر، اكتبلي CSS جاهز للعناصر الأساسية.
```

## 🧩 مواصفات النظام (مهمة — علشان الديزين يركّب على الموقع)

> ابعت الجزء ده كمان لـ Claude Design عشان يعرف حدود النظام.

**النظام بيدعم القيم دي جاهزة (لو الديزين استخدمها = يركّب فوراً بدون كود):**

| العنصر | القيم المتاحة |
|--------|----------------|
| شكل الكروت | `solid` / `glass` / `outline` |
| النافبار | `blur` / `solid` / `transparent` |
| الأزرار | `rounded` / `sharp` / `pill` |
| تخطيط الـ Hero | `centered` / `split` / `massive` / `cover-full` / `minimal` |
| شبكة المشاريع | `grid` / `masonry` / `list` |
| الخلفية | `dark` / `ocean` / `sunset` / `forest` / `mono` / `pearl` (أو HEX مخصص) |
| الحركة | `fade` / `fade-up` |

**ملاحظة:** لو الديزين فيه أنماط جديدة خارج القيم دي (شكل كارت مبتكر، hero فريد) — مفيش مشكلة، أنا هضيف CSS بسيط ليها. القائمة دي بس عشان لو حبيت أسرع تركيب.

## ✅ الخلاصة

1. ابعت **البرومبت + مواصفات النظام** لـ Claude Design
2. استلم منه **الصورة + الألوان + الخطوط + القرارات (+ CSS لو أمكن)**
3. ابعتهم لي
4. أنا أحوّلهم لثيم شغّال → يظهر في الداش بورد → تختاره → live

**نصيحة:** قوله **"اختار اتجاه واحد جريء ونفّذه بجرأة، متبقاش محافظ"** — عشان تطلع بثيم صادم فعلاً مش مجرد تغيير ألوان.
